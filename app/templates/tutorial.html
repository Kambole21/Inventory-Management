2. How to record a new inventory🖋️

a. A new worksheet of a blank inventory (Image 2.0).
Step 1 You will have to enter the the Department or School you are recording inventories from.
Step 2 You are required to enter the Date when you are recording the inventory.
The inventory worksheet has action buttons (Add Row, Submit Inventory, Export To CSV, and Delete).
Start with the Add Row button, it adds a row to the worksheet with the aim that the user can record a number of equipments without starting a new inventory.
If a user has entered undesired records, you can delete the entire row by pressing 🗑️

lets go to the next step

Image 2.0

b. The worksheet has eight (8) columns, and each column of the inventory worksheet is associated to one row. This can be done as follows:
⚫ No column automatically assigned when you add a row by pressing the Add Row button
⚫ USERNAME column contains the names of the individual using the equipments. In case an equipment is used by at least two people, you are required to enter Office
⚫ POSITION column requires you to enter the name of the office where you are excersing your inventories from
⚫ ICT EQUIPMENT column contains pre-defined equipment types that you can select from, but if you are instructed to enter an equipment which is not on the list provided you can type in the the equipment name
⚫ MODEL DETAILS column requires you to enter the model details of the equipment.
Note That 📌
Equipments with the same product and manufacturer can have the same model details
⚫ SERIAL NUMBER column requires you to enter the serial number of the equipment
⚫ STATUS column contains list of status that you are required to select from. which are as follows:
🔹 Working: equipment has no problems
🔹 Faulty: equipment has a problem
🔹 Absolute: equipment needs to be replaced
⚫ COMMENT column requires you to pass a comment on what needs to be done on the equipment. If the equipment has no problems or nothing has been said about it from the user, you can leave the section empty
